# What is this?
This is a desktop app for Re:Dollars, a garage of Bangumi which can be found at [here](https://bgm.tv/dev/app/4337), and it is still being developing, and extremely rough and simple.

# What can you do?
Because the project is currently too rudimentary, you can hardly use it normally. However, you can still submit pull requests or raise issues to help this project develop.

# Repository operator's words
This project was started solely for my personal learning of HTTP development and to further familiarize myself with the imgui framework. Thanks to all the framework authors, API providers as referenced in the project and bgmers who helped build this repository.
